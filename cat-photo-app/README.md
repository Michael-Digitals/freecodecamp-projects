# Cat Photo App
This is a simple webpage built with HTML to practice basic elements like headings, paragraphs, images, links, and lists.

## Features
- Ordered and unordered lists
- Clickable links
- Embedded images

## Technologies Used
- HTML5

## How to Run
1. Clone this repository
2. Open 'index.html' in your browser
